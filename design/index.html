Take-off Design Mockups

Interactive HTML mockups for evaluating layout options, colour schemes, and interaction patterns before writing production code. Use the theme switcher on each page to toggle between Light, Dark, and Brand colour schemes.

Concept 1 — Sidebar + Canvas In progress

Dark top bar with inline job name and three-tab navigation. Fixed 284 px sidebar holds a six-mode switcher (Legend, Count, Annotate, Measure, Area, Erase) with context-sensitive panels below it. The drawing canvas fills the remaining space with floating page-nav, zoom controls, and minimap overlays. Annotations Report and Cost Table are full-page scrollable views sharing the same top bar.

Concept 2 — Canvas-dominant Tool Rail In progress

Slim 36 px header bar with inline page navigation and scale selectors. A 52 px icon-only tool rail replaces the fixed sidebar. Mode controls float over the canvas in a draggable panel that can be collapsed to a pill. Annotations and Cost Table open as 400 px right drawers — the drawing stays visible. Data entry (room names, measurements, areas) slides up as a bottom sheet rather than a centred modal.

Concept 3 — Section Nav + Context Panel In progress

VS Code / Notion-inspired three-column shell: a 60 px icon-only section nav, a 260 px context panel whose content changes entirely per section, and an uninterrupted main canvas or report view. No top bar — job name, page navigation, tool palette, minimap, and zoom all live in the context panel. Data entry (room names, measurements, areas) and destructive confirmations happen inline inside the panel rather than in modals or overlays.
